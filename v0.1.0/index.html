pfd-kit documentation

PFD-kit version 0.7.0 documentation

PFD-kit is a cloud-based workflow written in Python for fine-tune and distillation of pre-trained atomic models. The main features of PFD-kit include automated data generation, model training, and evaluation, making it a powerful tool for researchers in the field of computational materials science.
